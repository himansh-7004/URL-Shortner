For running this project. we need to follow these steps.
step 1 : need to make and activate virtual environment.
step 2 : install django 
step 3 : run python manage.py runserver
step 4 : call this api on postman to short url and it will give you resulted shorted url  - http://localhost:8000/?url=url-link 
step 5 : hit shorted url directly on postman then you will be redirected to original url in browser.
